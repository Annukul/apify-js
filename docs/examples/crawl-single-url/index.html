Crawl a single URL

This example uses the Apify.utils.requestAsBrowser() function to grab the HTML of a web page.

const Apify = require('apify');

Apify.main(async () => {
    // Get the HTML of a web page
    const { body } = await Apify.utils.requestAsBrowser({ url: 'https://www.example.com' });
    console.log(body);
});

If you don't want to hard-code the URL into the script, refer to the Accept User Input example.
