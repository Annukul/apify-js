StealthOptions

Configuration of stealth tricks for a proper hiding effect all of them should be set to true. These tricks are applied only when the stealth option is set to true.

Properties

ParamTypeDefault
[addPlugins]booleantrue

If plugins should be added to the navigator.

[emulateWindowFrame]booleantrue

Emulates window Iframe.

[emulateWebGL]booleantrue

Emulates graphic card.

[emulateConsoleDebug]booleantrue

Emulates console.debug to return null.

[addLanguage]booleantrue

Adds languages to the navigator.

[hideWebDriver]booleantrue

Hides the webdriver by changing the navigator proto.

[hackPermissions]booleantrue

Fakes interaction with permissions.

[mockChrome]booleantrue

Adds the chrome runtime properties.

[mockChromeInIframe]booleantrue

Adds the chrome runtime properties inside the every newly created iframe.

[mockDeviceMemory]booleantrue

Sets device memory to other value than 0.
